How to configure Google Maps

Please note

If Google Map is not showing up or requires API key (after you have performed the changes as explained below), please read this tutorial on how to get your API key and how to perform changes: latecnosfera.com/2016/06/google-maps-api-error-missing-keymap-error-solved.html.

Please go to assets/scripts/main.js and look for this piece of code google.maps.LatLng(44.5403, -78.5463). Replace the numbers with Latitude and Longitude of your location. You can convert your address to LatLng using this website: www.latlong.net. Please make sure you keep the format like this: (44.5403, -78.5463) - so the numbers are divided by comma and space.
